# Planning 문서 안내

## 목적
이 폴더는 개발 계획, 진행 상황, 완료 보고를 정리합니다.

## 구조
- `1_inspection/`: Inspection System 관련 계획/진행 문서
- `2_comparison/`: Comparison System 관련 계획/진행 문서
- `archive/`: 완료/보고 문서 보관
- `ACTIVE_PLANS.md`: 현재 우선순위와 참조 문서

## 운영 원칙
- 진행 중 문서는 `1_inspection/`, `2_comparison/`에 유지합니다.
- 완료된 보고서는 `archive/`로 이동합니다.
- 현재 업무 흐름은 `ACTIVE_PLANS.md`를 기준으로 합니다.
